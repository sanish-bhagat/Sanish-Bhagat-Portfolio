# Sanish Bhagat — Portfolio

A personal portfolio website built with pure HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://sanish-bhagat.github.io/Sanish-Bhagat-Portfolio/

---

## About

This is my personal portfolio showcasing my AI/ML projects, skills, certifications, and education. No frameworks or build tools — just vanilla HTML, CSS, and JS.

## Features

- Dark / Light mode toggle
- Responsive on all screen sizes
- Animated hero section with particle background
- Filterable project cards
- Certifications section
- Animated skill bars and scroll reveal effects
- Contact form with validation

## Sections

- Hero
- About
- Skills
- Projects
- Education
- Certifications
- Testimonials
- Contact

## Getting Started

```bash
git clone https://github.com/sanishbhagat/portfolio.git
cd portfolio
```

Then just open `index.html` in your browser. No installs needed.

## Customization

All personal data (name, email, projects, certs) is stored in the `<script>` section at the bottom of `index.html` and is easy to edit.

To add a profile photo, replace the avatar icon with an `<img>` tag pointing to your image.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts — Bebas Neue, Syne, JetBrains Mono
- Font Awesome 6 (icons, via CDN)
